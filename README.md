# DiscordBotReplit
A Discord bot. Deployed on Replit. 
https://replit.com/@mimiyayaya/BottheBot#deploy-commands.js


FUNCTIONS:

/ping
-replies with pong for testing.

/server
-display server info.

/spam [input]
-spam something 3 times.

/singbday [input]
-sing the birthday song to someone.

/flipcoin
-flip a coin.
